# これはなに?

- 税金を正しく把握するための計算ツール群の置き場所
  - 個人事業主向けふるさと納税計算機
  - 免税事業者と簡易課税事業者の比較計算機をリリース
- https://money-hacker.web.app/

# アーキテクチャ

- Flutter Web + Firebase Hosting + Github Actions
